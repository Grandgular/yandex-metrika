Демонстрация директив Яндекс.Метрики

ymGoal - Отслеживание целей (reachGoal)

Простая цель

<button [ymGoal]="'button_click'">Нажми меня</button>

Цель с параметрами

<button [ymGoal]="'purchase'" [ymParams]="{{ '{' }} product: 'iPhone', price: 999 {{ '}' }}"> Купить iPhone </button>

Цель с событием наведения

Наведи на меня
<div [ymGoal]="'card_hover'" [ymEvent]="'mouseenter'"> Наведи на меня </div>

Цель с определенным счетчиком

<button [ymGoal]="'conversion'" [ymCounterId]="123456" [ymParams]="{{ '{' }} source: 'banner' {{ '}' }}"> Конверсия </button>

Условный трекинг

<button [ymGoal]="'premium_action'" [ymCondition]="isPremiumUser"> Премиум функция </button>

ymHit - Виртуальные просмотры страниц (hit)

Виртуальная страница

<button [ymHit]="'/virtual/modal-success'" [ymEvent]="'click'"> Открыть модальное окно </button>

Хит с параметрами

Переключить таб
<div [ymHit]="'/virtual/tab-content'" [ymParams]="{{ '{' }} tab: 'profile' {{ '}' }}"> Переключить таб </div>

ymParams - Параметры визита

Параметры при клике

<button [ymParams]="{{ '{' }} button_clicked: true, section: 'navigation' {{ '}' }}"> Отследить клик </button>

Параметры при скролле

Прокрути меня
Контент для скролла...
<div [ymParams]="{{ '{' }} section_scrolled: true {{ '}' }}" [ymEvent]="'scroll'"> Прокрути меня </div>

ymNotBounce - Отмена отскока

Пользователь активен

<button [ymNotBounce] [ymEvent]="'click'"> Я активно пользуюсь сайтом! </button>

Отслеживание активности

Подвигайте мышкой в этой области
<div [ymNotBounce] [ymEvent]="'mousemove'"> Подвигайте мышкой в этой области </div>

ymMethod - Универсальный метод

Пользовательские параметры

<button [ymMethod]="'userParams'" [ymParams]="{{ '{' }} user_action: 'download' {{ '}' }}"> Скачать PDF </button>

Файловая загрузка

<button [ymMethod]="'file'" [ymParams]="'/documents/price-list.pdf'"> Скачать прайс-лист </button>

Формы и инпуты

Отправка формы

<form [ymGoal]="'form_submitted'" [ymEvent]="'submit'"> <!-- поля формы --> <button type="submit">Отправить</button> </form>

Поисковый запрос

<input [ymGoal]="'search_performed'" [ymEvent]="'input'" [ymParams]="{{ '{' }} type: 'search' {{ '}' }}" placeholder="Поиск..." />

Фокус на поле

<input [ymGoal]="'input_focused'" [ymEvent]="'focus'" [ymParams]="{{ '{' }} field: 'email' {{ '}' }}" placeholder="Введите email" />
